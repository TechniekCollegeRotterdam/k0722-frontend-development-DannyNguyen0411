About me

Hallo, ik ben Danny Nguyen, ik ben 17 jaar oud en ik doe nu de opleiding Software developer op het Techniek College.
Op deze school heb ik geleerd om met html5, css3, php en javascript werken.
Ik had ook geleerd om met de scrum methode, front-end en back-end te werken.


Mijn eigenschappen is creativiteit, nauwkeurigheid, gemotiveerd, leergierig en kalm.
Met deze eigenschappen heb ik een doorzettende vermogen om hard te werken.











Project

Op deze school moet ik elke periode een project maken.
Bij project moet je altijd gaan samen werken, want als je niet samenwerkt, kun je het ook geen project noemen.



Dit is mijn eerste project. Ik moest in 6 weken een website bouwen uit html en css.
Ik moet ook de html en css checken of er een error staat. De 3 weken die over zijn gebleven,
moeten we een bedrijfswebsite bouwen. We mogen kiezen voor welk bedrijf we de website bouwen.
Voor mijn eerste project heb ik een 8.1 gehaald.
